Turkish Airlines Sound Set for 3jCabin-Sounds - Sound Engine Plugin

REQUIREMENTS
-----------------------------------------------
FlyWithLua - https://forums.x-plane.org/index.php?/files/file/35579-flywithlua-core-edition-for-x-plane-11-windows-linux-mac-os-x-version/
3jLib1 UI Extension Library - https://forums.x-plane.org/index.php?/files/file/34720-3jlib1-ui-extension-library-for-flywithlua/
3jCabin-sounds Sound Engine Plugin - https://forums.x-plane.org/index.php?/files/file/34723-3jcabin-sounds-sound-engine-plugin/

INSTALL
-----------------------------------------------
Put the folder "3jCabin-sounds" to the FlyWithLua-Scripts folder
(".../X-Plane 11/Resources/plugins/FlyWithLua/Scripts/")
Run X-Plane
Move the mouse to the bottom center of the screen, a button to open the sounds window appears
Click on it, the sounds window will open (You can also apply a key or controller button to open it)
Or sounds will play automaticaly :)

KURULUM
-----------------------------------------------
"3jCabin-sounds" klasörünü FlyWithLua-Scripts dizinine atın
(".../X-Plane 11/Resources/plugins/FlyWithLua/Scripts/")
X-Plane'i açın
Fareyi ekranın alt orta kısmına getirin, seslerin olduğu pencereyi açmak için bir düğme görünecektir
Düğmenin üzerine tıklayın, seslerin olduğu pencere açılır (X-Planeay arlarından bu işlem için bir kısayol atayabilirsiniz.)
Ya da seslerin otomatik olarak oynamasını bekleyebilirsiniz.
